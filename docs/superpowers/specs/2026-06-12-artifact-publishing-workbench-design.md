# 2026-06-12 产物预览与发布工作台设计

## 背景

当前控制台右侧只有产物列表和简短摘要。用户能打开文件，但还不能直接在控制台内完成“预览最终视频、区分版本、拿走发布文案”这套日常动作。

## 本次范围

围绕 `P1-3` 补齐一个最小但可用的工作台：

1. 右侧内嵌最终视频播放器，默认播放当前正式版本，没有正式版本时退回最新版本。
2. 摘要区展示封面、当前视频时长、文件大小、正式版本名和版本列表。
3. 发布标题、标签、描述提供一键复制。
4. 版本列表明确标注当前正式版本。

## 设计取舍

- 不增加新页面，直接复用右侧工作台。
- 视频时长优先由后端用 `ffprobe` 读取；失败时返回空值，前端显示 `-`，不阻塞主流程。
- 复制动作在前端完成，优先使用 `navigator.clipboard`，失败时回退到 `window.prompt`。

## 验收点

1. 完成任务后，控制台内可直接播放最终 mp4。
2. 多个版本能在版本列表中清晰区分，且当前正式版本有明显标识。
3. 发布标题、标签、描述能直接复制，不需要打开 JSON 文件。
