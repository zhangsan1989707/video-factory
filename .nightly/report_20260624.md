## 暗色模式设计说明

- 基于 CSS 变量体系，与现有主题无缝集成
- 支持三种模式：跟随系统 (`prefers-color-scheme`)、手动浅色、手动深色
- 选择持久化到 `localStorage`，跨会话保持
- 暗色配色方案：深蓝灰底色 (#0f1117)，柔和对比度，适合长时间编码/审阅使用
